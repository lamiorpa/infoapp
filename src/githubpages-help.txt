https://dev.to/yuribenjamin/how-to-deploy-react-app-in-github-pages-2a1f

npm init react-app my-app

cd my-app
npm install gh-pages --save-dev

Top level add:
"homepage": "http://{username}.github.io/{my-app}" // app name

scripts add:
"predeploy": "npm run build",
"deploy": "gh-pages -d build"

Create new github repository with same name as your app

// initialize and create remote git branch
git init 
git remote add origin https://github.com/{username}/{my-app}.git

npm run predeploy // optional
npm run deploy

-----

// add your source code to git, instead of just build software
git add .
git commit -m "Your awesome message"
git push origin master

-----

dependencies:

npm install --save react-router-dom bootstrap
npm install --save jquery popper.js
